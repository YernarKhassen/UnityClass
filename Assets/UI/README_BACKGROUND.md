# Фоновое изображение для меню

## Текущая настройка
В сцене `main.unity` уже добавлен элемент `BackgroundImage`, который отображает темно-синий фон.

## Как добавить собственное изображение

### Вариант 1: Использовать готовое изображение
1. Найдите подходящее изображение (рекомендуется 1920x1080 или выше):
   - Темный фон с яркими акцентами (оранжевый, красный, желтый)
   - Тематика: взрывы, бомбы, пиксельный стиль
   - Бесплатные источники:
     - https://opengameart.org
     - https://itch.io/game-assets
     - https://www.kenney.nl/assets

2. Сохраните изображение в формате PNG или JPG в папку `Assets/UI/`

3. В Unity:
   - Выберите объект `BackgroundImage` в иерархии сцены `main`
   - В Inspector найдите компонент `Image`
   - В поле `Source Image` перетащите ваше изображение
   - Убедитесь, что `Color` установлен на белый (255, 255, 255, 255) для полной яркости

### Вариант 2: Создать изображение программно
Если у вас установлен Python с библиотекой Pillow:
```bash
pip install Pillow
python create_background.py
```

### Вариант 3: Использовать текущий цветной фон
Текущий фон использует темно-синий цвет. Вы можете изменить цвет в Inspector компонента `Image` на объекте `BackgroundImage`.

## Рекомендуемые цвета для фона
- Темно-синий: RGB(26, 26, 46) - текущий
- Темно-фиолетовый: RGB(30, 20, 40)
- Темно-красный: RGB(40, 20, 20)
- Черный с градиентом: RGB(10, 10, 15) → RGB(20, 20, 30)
